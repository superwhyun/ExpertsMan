# 전문가 목록 관리 시스템

전문가 목록을 관리하고 세미나 발표를 요청하기 위한 웹 애플리케이션입니다.

## 기능

### 관리자 기능
- 전문가 목록 CRUD (이름, 소속, 직위, 이메일, 전화번호, 자문료)
- 전문가별 고유 입력 링크 생성 (랜덤 ID + 6자리 비밀번호)
- 생성된 링크 및 비밀번호 조회/공유

### 전문가 입력 기능
- 비밀번호 보호된 입력 페이지
- **발표자 인적사항**: 성명, 주민번호, 주소, 소속, 부서, 전화, 이메일
- **학력**: 학위취득년도, 학위, 학교 (동적 추가 가능)
- **경력**: 기간, 기관, 업무 (동적 추가 가능)
- **주요업적**: 수상이력, 전문기술분야 등
- **발표요약**: 강연제목, 소제목 5개, 요약(500자 내외)
- **개인정보 동의서**: 웹 서명 기능 포함
- **임시저장**: 브라우저 localStorage에 자동/수동 저장
- **PDF 다운로드**: 3페이지 PDF 생성 (html2canvas + jspdf)

## 기술 스택
- **Framework**: React + Vite
- **Styling**: Tailwind CSS v4
- **Routing**: React Router DOM
- **PDF**: html2canvas + jspdf
- **Storage**: localStorage (서버리스)

## 사용 방법

### 관리자
1. `/` 또는 `/admin` 접속
2. "전문가 추가" 버튼 클릭
3. 전문가 정보 입력 후 저장
4. 생성된 링크와 비밀번호를 전문가에게 전달

### 전문가
1. 관리자로부터 받은 링크 접속
2. 비밀번호 입력 후 접속
3. 인적사항, 발표요약, 개인정보 동의서 순서대로 입력
4. "임시저장" 버튼으로 중간 저장 가능
5. 모든 항목 입력 완료 후 "PDF 다운로드" 버튼 클릭

## 라우팅
- `/` - 관리자 페이지 (전문가 목록)
- `/form/:expertId` - 비밀번호 확인 페이지
- `/form/:expertId/edit` - 전문가 입력 페이지

## 데이터 저장
- 모든 데이터는 브라우저의 localStorage에 저장됩니다
- 서버는 아묟ㄴ 정볼 저장하지 않습니다 (완전 서버리스)
- 임시저장 데이터는 30초마다 자동 저장됩니다

## PDF 생성
- 3페이지 구성:
  - 페이지 1: 발표자 인적사항 (기본정보, 학력, 경력, 주요업적)
  - 페이지 2: 발표요약 (강연제목, 소제목, 요약)
  - 페이지 3: 개인정보 수집 및 이용 확인서 (동의서 + 서명)

## 주의사항
- 브라우저 캐시 삭제 시 저장된 데이터가 삭제될 수 있습니다
- 중요한 데이터는 PDF로 즉시 다운로드 받아 보관하세요
